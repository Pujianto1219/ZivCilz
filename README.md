UDP server installation for ZIVPN Tunnel (SSH/DNS/UDP) VPN app.
<br>

>Server binary for Linux amd64 and arm.

### INSTALL
```
apt update && apt install -y wget && wget https://raw.githubusercontent.com/Pujianto1219/ZivCilz/main/setup.sh && chmod +x setup.sh && ./setup.sh
```
#### Installation AMD
```
wget -O zi.sh https://raw.githubusercontent.com/Pujianto1219/ZivCilz/main/zi.sh; sudo chmod +x zi.sh; sudo ./zi.sh
```

#### Installation ARM
```
bash <(curl -fsSL https://raw.githubusercontent.com/zahidbd2/udp-zivpn/main/zi2.sh)
```


### Uninstall

```
sudo wget -O ziun.sh https://raw.githubusercontent.com/Pujianto1219/ZivCilz/main/uninstall.sh; sudo chmod +x ziun.sh; sudo ./ziun.sh
```
